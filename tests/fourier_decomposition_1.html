About the Visualization

These plots show the Fourier series approximation of different functions. One can select a function, which is shown as a dashed line in Plot 1 (left), and generate a Fourier series of the function (shown as cyan line in Plot 1) by setting the number of terms, \(N\), to include in the series. In Plot 2 (right) the amplitude the highest order term, \(N\), is shown.


Plot 1 (left):
  • (dashed line) the function that is chosen to be approximated.
  • (blue line) the sum of \(N\) terms in the Fourier series approximation (where \(N\) is chosen by the user).

Plot 2 (right):
  • (blue line) the \(N^\mathrm{th}\) term in the Fourier series

Drop-down and Sliders:
  • Drop-down: select the function to be approximated
  • Slider: choose the number of terms to include in the Fourier series; the amplitude of the \(N^\mathrm{th}\) term will be displayed in Plot 2.
