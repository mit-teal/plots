About the Visualization

These plots show the normal mode solution for waves on an infinite string, of the form \(A(x,t)=A_{m}\sin(k_{m}x + \alpha_{m})\sin(\omega_{m}t + \beta_{m})\). The left plot shows the solution at a fixed time \(t=t_{0}\) (as a function of position \(x\)) and the right plot shows the solution at a fixed position \(x=x_{0}\) (as a function of time \(t\)).


Plot 1 (left):
  • (blue line) \(A(x,t)=A_{m}\sin(k_{m}x + \alpha_{m})\sin(\omega_{m}t + \beta_{m})\) is plotted as a funciton of \(x\), at a fixed time \(t_{0}\)
  • (red dashed line) the position \(x_{0}\) is shown
  • (red dot) the amplitude \(A(x_{0},t_{0})\) is shown
  • the graph changes when \(t_{0}\) is varied

Plot 2 (right):
  • (red line) \(A(x,t)=A_{m}\sin(k_{m}x + \alpha_{m})\sin(\omega_{m}t + \beta_{m})\) is plotted as a function of \(t\), at a fixed position \(x_{0}\)
  • (blue dashed line) the position \(t_{0}\) is shown
  • (blue dot) the amplitude \(A(x_{0},t_{0})\) is shown
  • the graph changes when \(x_{0}\) is varied

Sliders:
  • \(x_{0}\): vary the position at which the amplitude is "observed"; this will change the vertical dashed line in Plot 1 and the shape of the wave in Plot 2.
  • \(t_{0}\): vary the time at which the amplitude is "observed"; this will change the vertical dashed line in Plot 2 and the shape of the wave in Plot 1.

Ideas for Exploration

Consider the following questions and possible actions:

  • How does the amplitude change as a function of time, for fixed position?
  • How does the amplitude change as a function of position, for fixed time?
